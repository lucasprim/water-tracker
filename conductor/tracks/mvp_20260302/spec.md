# Specification: Water Tracker MVP

**Track ID:** mvp_20260302
**Type:** Feature
**Created:** 2026-03-02
**Status:** Complete

## Summary

Build a minimal but complete, visually polished Water Tracker app for macOS: a
persistent menu bar app with an animated water-level UI, continuous non-exclusive
webcam monitoring to detect drinking, a countdown timer, and gentle alerts when
the user forgets to hydrate.

## Context

Water Tracker is a lightweight, privacy-first native Mac app. No backend, no
accounts — all data lives on-device via SwiftData. Targets macOS 14 (Sonoma)+
using SwiftUI, MenuBarExtra, UserNotifications, and Vision/AVFoundation.

## User Story

As a Mac user trying to build a daily hydration habit, I want a visually
delightful menu bar app that watches for me drinking water and shows my progress
as a beautiful animated cup — so I stay motivated without interrupting my work.

## Acceptance Criteria

- [x] App lives in the menu bar; clicking the icon opens a popover/panel
- [x] The popover features a realistic animated cup showing the current hydration
      level as a rising water fill (with subtle sloshing water animation)
- [x] A "Log Bottle" button records a water entry using the configured bottle
      size; the cup animates to the new fill level
- [x] Daily water goal is configurable (ml); the cup fill % reflects
      logged-ml / goal-ml
- [x] The webcam runs continuously in the background using non-exclusive camera
      access (Zoom/FaceTime can use the camera simultaneously)
- [x] A countdown timer is visible in the menu bar; the user configures the
      max drink interval (default: 15 min)
- [x] When drinking is detected by the webcam, the timer resets and the menu bar
      icon turns blue momentarily (bottle logging is manual only per design decision)
- [x] If the timer expires, a system alert notification is shown
- [x] Monitoring and the timer stop automatically once the daily goal is met
- [x] A settings screen (clean, native macOS style) allows configuring: bottle
      size, daily goal, and max drink interval
- [x] Overall visual design is polished: smooth animations, consistent spacing,
      clear typography

## Dependencies

None — greenfield project.

## Out of Scope

- CloudKit / iCloud sync
- History, charts, or analytics views
- Apple Watch / iPhone companion app
- Advanced ML / custom Vision models (basic Vision heuristics for MVP)

## Technical Notes

- Swift + SwiftUI + SwiftData + MenuBarExtra, macOS 14+
- Animated water cup: implement using SwiftUI Canvas + sine-wave Path animation
  (no SpriteKit or Metal required for MVP)
- Camera: non-exclusive AVCaptureSession at low resolution/frame-rate; Vision
  framework detects drinking gesture (hand + bottle near face heuristic)
- Timer + detection logic are decoupled from the UI layer
- All data on-device; no network calls
